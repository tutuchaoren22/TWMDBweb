## 如何运行项目

1. 直接访问网页`http://twmdb.twogoats.fun:8080/`；

  或者

1. 下载项目源码，运行`TWMDBweb`类中的`main`函数;

2. 在浏览器访问`localhost:8080`，进入影院首页，通过点击电影图片或搜索电影片名来进入详情页及搜索页面；
